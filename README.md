# 오라클 프로젝트

### 📌 Summary
교육센터 운영프로그램을 위한 데이터베이스 설계를 의뢰 받아,

요구 분석 및 DB 설계/구현 을 진행한 프로젝트입니다.

---

### 💡 주요기능
1. 계정별 데이터베이스 설계 (관리자 / 교사 / 교육생)
2. 과목 / 과정별 관리 구조 설계
3. 자동 업데이트 기능 위한 프로시저 / 트리거 생성
	- 요구 사항 반영 + 사용자 편의성 증가를 고려함

----

### 🔨 Tech Stack
1. ANSI-SQL  
2. Oracle(PL/SQL)
